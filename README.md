# CPIT-770 -  A* Search Python code
## A* Pathfinding (Python)

This repository contains simple Python implementations of the **A\*** search algorithm.

- **astarclaude.py**  
  A quick implementation of the A\* algorithm focusing on core logic.
  
- **interactive.py**  
  An interactive grid-based A\* pathfinding visualizer using Tkinter.
